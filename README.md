# 🎓 AI Automated Timetable Generator

A web-based timetable generation system developed using Python and Flask to automate the scheduling process and reduce manual effort in educational institutions.

## 🚀 Features

* Automatic timetable generation
* Faculty allocation management
* Subject scheduling
* Conflict-free timetable creation
* Dynamic timetable display
* Database integration
* User-friendly web interface

## 🛠️ Tech Stack

### Backend

* Python
* Flask

### Frontend

* HTML5
* CSS3
* Jinja Templates

### Database

* SQLite

## 🎯 Project Objective

The objective of this project is to automate the timetable generation process by intelligently assigning subjects, faculty members, and time slots while minimizing scheduling conflicts.

## 💡 Key Highlights

* Reduces manual timetable preparation effort.
* Improves scheduling efficiency.
* Generates structured timetables automatically.
* Easy to modify and scale for future requirements.

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/malavkeshav1025/ai-automated-timetable-generator.git
```

2. Navigate to project folder

```bash
cd ai-automated-timetable-generator
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```
